# Bienvenida al proyecto

¡Bienvenido/a al proyecto **emailMK**! Este documento te guía para comenzar rápidamente.

## Objetivo
Este proyecto inicia un espacio para construir, documentar y evolucionar el trabajo de forma organizada.

## Inicio del proyecto
Este punto marca el inicio del proyecto tras la limpieza inicial del repositorio en GitHub.

## Estructura del repositorio
- `app/`: código de la aplicación (servicios, controladores, etc.).
- `config/`: configuración del proyecto (por ejemplo, base de datos).
- `database/`: esquema SQL y recursos de base de datos.
- `public/`: recursos estáticos y assets públicos.
- `login.php` y `dashboard.php`: entradas principales de autenticación y panel.

## Requisitos
- PHP 8.x con extensiones habituales (PDO para MySQL).
- MySQL 8.x (o compatible).

## Configuración de base de datos
1. Revisa las credenciales en `config/database.php`.
2. Crea la base de datos y aplica el esquema en `database/schema.sql`.

## Inicio rápido
1. Configura la base de datos.
2. Levanta un servidor PHP local apuntando a la raíz del proyecto.
3. Accede a `/login.php` para iniciar sesión.

## Primeros pasos
1. Revisa la estructura del repositorio y el archivo `index.php`.
2. Define los objetivos iniciales y anota las tareas prioritarias.
3. Establece criterios de calidad (estilo de código, pruebas, y documentación).

## Colaboración
- Mantén los cambios pequeños y bien descritos.
- Documenta las decisiones relevantes en este archivo.

## Contacto
Si tienes dudas o sugerencias, agrega un comentario en el historial del proyecto para que el equipo pueda seguir el contexto.
